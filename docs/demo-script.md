# 面试展示脚本

## 1. 打开首页

访问 `http://127.0.0.1:5173/`，说明项目定位：面向在日外国留学生的 AI 生活支援系统，整体采用政府门户风格。

## 2. 展示咨询分类

说明系统覆盖在留手续、生活指南、工作打工、奖学金、医疗保险、税金、学校生活、紧急支援等场景。

## 3. 打开 /chat 提问

进入 `/chat`，选择“工作·打工”，输入：

```text
留学生可以每周打工多少小时？
```

展示知识库回复、参考资料和安全提示。

## 4. 展示参考资料

说明回答正文和参考资料分离，前端最多显示 3 条来源卡片，避免把来源拼接进回复正文。

## 5. 打开知识库管理

进入 `/admin/knowledge`，展示知识库列表、分类筛选、关键词搜索、新增、编辑和删除。

## 6. 新增知识

新增一条生活指南知识，例如“搬家后的住址变更”，说明后端暂不使用数据库保存知识库，而是使用 JSON 文件作为轻量 MVP。

## 7. 上传 TXT/PDF

进入 `/admin/documents`，上传 TXT 文件：

```text
搬家后，通常需要在规定期间内到新的市区町村窗口办理住址变更手续。办理时可能需要在留卡、本人确认材料等。具体要求请以居住地自治体发布的信息为准。
```

展示上传成功、文件大小、导入 chunk 数量和上传时间。

## 8. 回到聊天页验证检索

回到 `/chat`，选择“生活指南”，输入：

```text
搬家后需要办什么手续？
```

展示导入资料已经可以被检索，并出现在参考资料中。

## 9. 打开数据统计页面

进入 `/admin/dashboard`，展示总咨询数、总消息数、知识库条目数、上传文档数、反馈数、分类统计和最近 5 条咨询记录。

## 10. 说明工程亮点

- 前后端分离
- 本地 RAG MVP
- PDF/TXT 资料导入
- 聊天历史与反馈闭环
- SQLite + JSON 混合数据存储
- Docker 基础配置
- README、API 文档和面试演示脚本完整
